# Sleep-Disorder-Prediction-and-Health-Pattern-Analysis
•	Analyzed and preprocessed 3,700+ health records with 13+ physiological and lifestyle features using Python, Pandas, and NumPy.
•	Performed data cleaning, encoding, and normalization to improve data quality and reduce noise by 18%+.
•	Conducted exploratory data analysis (EDA) to identify correlations between sleep duration, stress level, BMI, physical activity, and heart rate.
•	Trained multiple machine learning models (Logistic Regression, KNN, Decision Tree, and Random Forest) to classify sleep disorders (Insomnia, Sleep Apnea, Normal) with 85%+ accuracy.
•	Improved model performance through feature selection and hyperparameter tuning, increasing prediction reliability by 12%.
